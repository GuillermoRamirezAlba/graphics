What is Ray tracing?

Ray tracing is a rendering technique that aims to simulate the way light bounces off objects, in turn creating more realistic shadows, reflections, and lighting effects.

With the usage of C++ and OpenGL, I am able to render any shapes onto the screen. This includes disks, spheres and so much more.

This ray-tracer I created does not only render realistic images, it also is able to make a GIF animation of various screenshots within the scene.

Key aspects of my project include Specular reflections, highlights, sub-sampling and spherical texturing using Perlin Noise.
